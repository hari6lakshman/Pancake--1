# **App Name**: Pan cake

## Core Features:

- Interactive Chat: AI-powered chat interface for answering user questions, offering advice, and providing emotional support.
- Daily Mood Check-ins: Regular prompts for users to log their mood, with options for personalized encouragement and follow-up questions from the AI.
- Personalized Affirmations: AI tool for creating and delivering daily positive affirmations tailored to the user's current mood and needs.
- Coping Strategies Library: A curated list of supportive coping strategies and resources accessible to the user at any time.
- Encouragement Bot: Send custom messages tailored to the users current state.

## Style Guidelines:

- Primary color: Baby pink (#FFB7C5) to create a cheerful and inviting atmosphere.
- Background color: Very light baby pink (#FFD1DC), near-white to avoid overwhelming the user, yet retain warmth.
- Accent color: Sky blue (#87CEEB) for a sense of calm and balance, used sparingly for highlights and calls to action.
- Font pairing: 'Belleza' (sans-serif) for headlines and short chunks of text; 'Alegreya' (serif) for body text.
- Use simple, rounded icons in pastel colors to represent different features and emotions.
- Design a clean and user-friendly layout with ample spacing to avoid feeling cluttered.
- Incorporate subtle, gentle animations for feedback and transitions to enhance user experience without being distracting.
